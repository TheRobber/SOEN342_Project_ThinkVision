# SOEN342_Project_ThinkVision

SETUP INSTRUCTIONS:

1) Open terminal and make sure cd is set to backend with command: cd backend

2) Install node modules with command: npm install

3) Start system with command: npm run dev

4) Backend running at http://localhost:3001 should appear, click on the link and you will have access to the program

NOTE: On first run, the database will be automatically created and populated with routes from the CSV file.
This may take a few seconds. Subsequent runs will use the existing database.

DATABASE:
- Database file: backend/data/train_network.db
- To reset the database, delete the file and restart the server
- See DATABASE_DOCUMENTATION.md for complete schema details
